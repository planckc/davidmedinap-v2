# Fe en Acción

## Tecnología con Propósito de Fe

Construyendo puentes entre la fe y la tecnología para conectar comunidades cristianas globales.

### Visión
Usar la tecnología como herramienta para el Reino - conectando personas, facilitando el discipulado y amplificando el impacto de comunidades de fe alrededor del mundo.
